2) CSS3 transform property in detail.
Transform changes shape, size, and position of elements.

Types of Transform
* translate()
* rotate()
* scale()
* skew()

Translate
Moves element.
div {
transform: translate(50px, 20px);
}

Rotate
Rotates element.
div {
transform: rotate(45deg);
}

Scale
Changes size.
div {
transform: scale(2,1);
}

Skew
Tilts element.
div {
transform: skew(20deg);
}

Combined Transform
div {
transform: rotate(30deg) scale(1.5);
}

Advantages
* Animation support
* Modern UI design

3) Bootstrap with responsive design.
Bootstrap is a popular front-end framework developed by Twitter for designing responsive and mobile-first web applications. It contains predefined CSS classes, grid systems, components, and JavaScript utilities that make web development faster.

Features of Bootstrap
* Mobile-first design
* Responsive grid system
* Predefined CSS classes
* Ready-made UI components
* Browser compatibility

Bootstrap Grid System
Bootstrap divides page width into 12 columns.

Example
<div class="container">
<div class="row">
    <div class="col-md-6">Left Side</div>
    <div class="col-md-6">Right Side</div>
</div>
</div>

Explanation
container creates fixed-width layout
row creates horizontal row
col-md-6 occupies 6 columns each

Responsive Breakpoints
Bootstrap uses different screen sizes:
* col-sm -> small devices
* col-md -> medium devices
* col-lg -> large devices
* col-xl -> extra large devices

Bootstrap Components
Examples:
* Buttons
* Cards
* Navbar
* Alerts

Button Example
<button class="btn btn-primary">Submit</button>

Advantages
* Saves development time
* Easy responsive design
* Professional appearance

2) Scrum fundamentals.
Scrum is an Agile framework used to manage software development in short iterations called Sprints. It helps teams deliver working software continuously and quickly adapt to changing requirements. Unlike traditional models, Scrum focuses on collaboration, incremental delivery, and customer feedback.

Basic Principles of Scrum
* Work is divided into small increments
* Development happens in fixed time cycles called Sprints
* Team collaboration is essential
* Frequent review and improvement are performed

Sprint
A Sprint is a fixed time period, usually 2 to 4 weeks, during which a set of features is developed. At the end of each Sprint, a usable product increment is delivered.

Scrum Process Flow
1. Product requirements collected
2. Product backlog prepared
3. Sprint planning done
4. Development starts
5. Daily meetings conducted
6. Sprint review held
7. Sprint retrospective performed

Advantages of Scrum
* Faster delivery
* Easy change handling
* Better communication
* Higher product quality

Importance in Software Industry
Scrum is widely used because modern software projects need flexibility and quick customer response.

1) HTML5 syntax & attribute & events.
HTML5 is the latest version of HTML used to create web pages and web applications. It introduced simpler syntax, new semantic elements, multimedia support, and browser APIs.

HTML5 Syntax
HTML5 syntax is simple and begins with the document declaration:

<!DOCTYPE html>
<html>
<head>
<title>Sample Page</title>
</head>
<body>
<h1>Hello World</h1>
</body>
</html>

<!DOCTYPE html> declares HTML5 document type
<html> is root element
<head> contains metadata
<title> gives page title
<body> contains visible content

Attributes provide additional information to HTML elements.

Common attributes

<a href="home.html">Home</a>

required makes field mandatory

HTML5 Events

Common events

* onchange
* onmouseover
* onkeyup

Example

When user clicks button, message appears.

Advantages

* Better interaction

3) Web storage & web socket.
Web Storage
Web storage stores data inside browser.
Two types:
* localStorage
* sessionStorage

localStorage
Stores data permanently until deleted.
<script>
localStorage.setItem("name", "Ananya");
</script>
Retrieve:
<script>
document.write(localStorage.getItem("name"));
</script>

sessionStorage
Stores data only during browser session.
sessionStorage.setItem("user", "student");

Difference
* localStorage: Permanent, Large storage
* sessionStorage: Temporary, Session only

Web Sockets
Web sockets allow two-way communication between browser and server.

Example
let socket = new WebSocket("ws://server.com");
socket.onopen = function() {
};
socket.send("Hello Server");

Advantages
* Real-time communication
* Fast data transfer
* Used in chat apps

AJAX means Asynchronous JavaScript and XML. Used to update page without reloading.

3) Node JS & npm in detail.
Introduction to Node.js
Node.js is an open-source JavaScript runtime built on the V8 engine of Google. It allows JavaScript to run outside the browser. Traditionally JavaScript was used only in browsers, but Node.js made server-side JavaScript possible.

Features of Node.js
* Event-driven architecture
* Non-blocking I/O operations
* Fast execution
* Single-threaded but highly scalable
* Suitable for real-time applications

Example of Node.js Program
const http = require('http');
http.createServer((req, res)=> {
res.write("Hello Node.js");
res.end();
}).listen(3000);

Explanation
require('http') imports HTTP module
createServer() creates server
listen(3000) runs server on port 3000

What is NPM
NPM stands for Node Package Manager. It is used to install libraries and packages.

Common NPM Commands
* npm init
* npm install express
* npm install
* npm uninstall package-name

Example
npm install express
This installs Express.js for web server development.

Advantages
* Huge package ecosystem
* Easy dependency management
* Faster project setup
